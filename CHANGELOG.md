# Changelog

All notable changes for the Idi Delites website project.

## [Unreleased] - 2026-06-19

### HTML
- Added a dedicated contact page section for the V&A Waterfront map embed.
- Removed Century City map and related toggle controls to simplify the contact page.
- Added gallery markup for product images and a lightbox view.
- Structured form sections with clear `div` containers for contact and enquiry pages.
- Improved page layout with cleaner section grouping and consistent headings.

### CSS
- Added styles for gallery layout, lightbox overlay, and lightbox navigation.
- Added responsive styling for maps, gallery, forms, and mobile navigation.
- Cleaned up leftover map toggle CSS after removing Century City support.
- Added `.form-message` styles for success, error, and loading feedback.
- Updated visual styling for buttons, cards, and layout spacing.

### JavaScript
- Added live search filtering for product cards.
- Added order button click feedback.
- Implemented asynchronous form submission with simulated AJAX for contact and enquiry forms.
- Added inline form validation and form message handling instead of browser alerts.
- Added input restrictions for name, phone, product, and message fields.
- Added dynamic product data loading and automatic order button behavior.
- Added lightbox gallery behavior with keyboard navigation and click-to-close support.
- Added scroll-based fade-in animations and mobile menu toggle behavior.

### SEO / Project Configuration
- Added `robots.txt` to manage crawler access.
- Added `sitemap.xml` to help search engines understand site structure.

### Notes
- The changelog captures the main front-end improvements made during this project.
- This file is intended to summarize updates across HTML, CSS, and JavaScript work.
